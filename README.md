# Biosign Dashboard
A scoring tool for exoplanet biosignature likelihood using NASA Open Data.

## Project Description
The Biosign Dashboard is a research-oriented tool designed for the evaluation of exoplanetary candidates. By synthesizing transit data and planetary characteristics from open-access catalogs, this project aims to provide a standardized likelihood score for potential biological activity.

## Development Roadmap
- Integration of NASA Exoplanet Archive API (NExScI).
- Implementation of atmospheric composition scoring algorithms.
- Development of a web-based visualization interface.
